# Forensic-DNA-Profile-Matching-Using-CNNs
Custom 1D CNN for forensic DNA match prediction (Match vs. No Match) using STR data. Built an end-to-end ML pipeline including data preprocessing, normalization, model training, evaluation, and a Streamlit web app for real-time predictions. Achieved 86% test accuracy and AUC-ROC of 0.9433, outperforming SVM and Random Forest baselines.
